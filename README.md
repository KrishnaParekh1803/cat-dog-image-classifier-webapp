\# Cat vs Dog Image Classifier 🐶🐱



This project is a deep learning based web application that classifies images of cats and dogs using a Convolutional Neural Network (CNN). The model is trained on the Dogs vs Cats dataset and can predict whether an uploaded image contains a cat or a dog.



The application allows users to upload an image through a web interface and receive a prediction with confidence score.



**Technologies Used**



\- Python

\- TensorFlow / Keras

\- Streamlit

\- NumPy

\- Pillow

**How to Run the App**



1\. Clone the repository

git clone https://github.com/YOUR\_USERNAME/cat-dog-image-classifier.git





2\. Install dependencies

pip install -r requirements.txt





3\. Run the Streamlit app

streamlit run streamlit\_app.py





4\. Open the local URL shown in the terminal to use the application.
